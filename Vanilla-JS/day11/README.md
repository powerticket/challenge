# 조건

- Have a **reset** (C) button.
- Support **all** basic operations (+ , - , * , / )
- Support for **'equals'** ( = ) button.
- Allow value carrying. i.e 2 *2* 2 *2* 2 without pressing equals.
- Don't give up!
